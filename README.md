# Carzone

A web application for car listings and management, built with JavaScript, CSS, HTML, and Python.

## Tech Stack

- **Frontend**: JavaScript (49.8%), CSS (36.1%), HTML (13.2%)
- **Backend**: Python (0.9%)

## Getting Started

### Prerequisites

- Node.js and npm
- Python 3.x

### Installation

1. Clone the repository:
```bash
git clone https://github.com/sisaymerga/carzone.git
cd carzone
```

2. Install dependencies:
```bash
npm install
```

3. Set up the backend (if applicable):
```bash
pip install -r requirements.txt
```

### Usage

Start the development server:
```bash
npm start
```

## Project Structure

- `/` - Frontend files (HTML, CSS, JavaScript)
- `/` - Backend files (Python)

## Features

- Browse car listings
- Search and filter cars
- Responsive design
